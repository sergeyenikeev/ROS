# Технический стек PatrolBot

## Основной стек

- ОС: Ubuntu Server 24.04 LTS arm64 для Raspberry Pi.
- Middleware: ROS 2 Jazzy Jalisco.
- Язык прикладной логики: C++17.
- Сборка: `colcon` + `ament_cmake`.
- SLAM: `slam_toolbox`.
- Локализация: `AMCL`.
- Навигация: `Nav2`.

## Почему выбран именно этот стек

### Ubuntu Server 24.04 LTS

- актуальная LTS-база под Raspberry Pi;
- совместима с ROS 2 Jazzy;
- хорошо подходит для headless-развёртывания робота.

### ROS 2 Jazzy

- долгосрочная поддержка;
- зрелый стек для `Nav2`, `slam_toolbox`, `robot_state_publisher`;
- хорошая база для дальнейшего переноса на железо и CI.

### C++17

- достаточно современный стандарт для безопасной и читаемой реализации;
- полностью совместим с практикой ROS 2;
- не требует лишнего усложнения зависимостей по сравнению с C++20.

### slam_toolbox

- стандартный и хорошо документированный выбор для 2D SLAM;
- поддерживает асинхронный режим и удобное сохранение карты;
- подходит для LiDAR-ориентированной indoor-навигации.

### AMCL

- отделяет эксплуатационный режим локализации от режима построения карты;
- проще для сопровождения и поиска типовых решений;
- практичнее для робота, который строит карту один раз, а затем многократно патрулирует помещение.

### Nav2

- де-факто стандарт навигации в ROS 2;
- предоставляет готовые action, costmap, planner, controller и recovery-поведение;
- хорошо сочетается с отдельным прикладным mission manager.

### Собственный пакет базы вместо ros2_control в v1

- ускоряет разработку прототипа;
- уменьшает зависимость от конкретного моторного контроллера;
- упрощает mock-тестирование без железа.

Этот выбор зафиксирован как решение v1. Для следующего этапа предусмотрена возможная миграция на `ros2_control`, если аппаратная часть стабилизируется.

### Отказ от robot_localization в v1

- в текущем варианте нет второй независимой оценки позы, которая дала бы заметную пользу;
- для простого дифференциального привода без IMU можно начать с чистой колёсной одометрии;
- это уменьшает количество настраиваемых фильтров и снижает риск ложной уверенности в данных.
