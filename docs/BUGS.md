# Известные баги и ограничения PatrolBot

- [ТРЕБУЕТ ПРОВЕРКИ НА ЖЕЛЕЗЕ] Реальный LiDAR и моторный драйвер ещё не подключались; воспроизведение: отсутствует; workaround: использовать mock-профиль запуска; проверить: совместимость с конкретным оборудованием Raspberry Pi.
- [ТРЕБУЕТ ПРОВЕРКИ НА ЖЕЛЕЗЕ] Push в `git@github.com:sergeyenikeev/ROS.git` недоступен из текущей среды из-за отсутствия SSH-ключа; воспроизведение: `git ls-remote git@github.com:sergeyenikeev/ROS.git`; workaround: выполнить push вручную после настройки SSH; проверить: права доступа к репозиторию.
- [ИЗВЕСТНО] Launch-профили ссылаются на исполняемые файлы базовой платформы и mission manager, которые будут добавлены следующими этапами; воспроизведение: запуск `ros2 launch patrolbot_bringup *.launch.py` до завершения реализации; workaround: дождаться следующих коммитов; проверить: успешность полного запуска после добавления узлов.
