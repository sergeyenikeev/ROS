# Список функций PatrolBot

- [СДЕЛАНО] Базовый каркас ROS 2 workspace и пакетная структура проекта.
- [СДЕЛАНО] Описание робота в `URDF/Xacro` и базовая TF-цепочка `map -> odom -> base_footprint -> base_link -> laser_frame`.
- [СДЕЛАНО] Базовые launch-файлы `mock_bringup`, `hardware_bringup`, `slam`, `localization`, `navigation`, `patrol`.
- [СДЕЛАНО] Базовая платформа с ограничением скоростей, watchdog по `cmd_vel`, одометрией и публикацией `diagnostic_msgs/DiagnosticArray`.
- [СДЕЛАНО] Mock-драйвер моторов и mock-публикатор `LaserScan` для запуска без реального железа.
- [СДЕЛАНО] Подготовлены конфиги `slam_toolbox`, `AMCL`, `Nav2`, параметры costmap и тестовая карта.
- [СДЕЛАНО] Реализованы интерфейсы `PatrolStatus`, `StartPatrol`, `StopPatrol`.
- [СДЕЛАНО] Реализован `mission_manager` с YAML-маршрутами, отправкой целей в `Nav2`, повторами, таймаутами и публикацией статуса миссии.
- [СДЕЛАНО] Добавлены unit-тесты для валидации параметров, загрузчика маршрутов, интегратора одометрии и state machine.
- [СДЕЛАНО] Добавлены launch/integration тесты для `mock_bringup`, `slam`, `localization`, `navigation` и сценариев patrol через mock Nav2 action server.
- [СДЕЛАНО] Добавлены bash-скрипты сборки, запуска, тестирования, сохранения карты и проверки форматирования.
- [СДЕЛАНО] Подготовлены README, архитектурная документация, runbook, описание логирования и настройка Raspberry Pi.
- [В РАБОТЕ] Адаптация `serial`-драйвера под реальный моторный контроллер.
- [НЕ СДЕЛАНО] Полевая валидация на реальном Raspberry Pi, LiDAR и моторном контроллере.
- [ПЛАНИРУЕТСЯ] Переход на полноценный аппаратный драйвер или `ros2_control` после стабилизации электроники.
- [ПЛАНИРУЕТСЯ] Добавление IMU и повторная оценка необходимости `robot_localization`.
