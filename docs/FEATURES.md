# Список функций PatrolBot

- [СДЕЛАНО] Подготовлен полный ROS 2 workspace с пакетной структурой проекта `PatrolBot`.
- [СДЕЛАНО] Описан робот в `URDF/Xacro` и зафиксирована базовая TF-цепочка `map -> odom -> base_footprint -> base_link -> laser_frame`.
- [СДЕЛАНО] Реализованы launch-профили `mock_bringup`, `hardware_bringup`, `slam`, `localization`, `navigation`, `patrol`.
- [СДЕЛАНО] Реализована базовая платформа с приёмом `/cmd_vel`, ограничением скоростей, watchdog по командам движения, публикацией `/odom`, `joint_states` и `diagnostics`.
- [СДЕЛАНО] Добавлены mock-драйвер моторов и mock-публикатор `LaserScan` для запуска без реального железа.
- [СДЕЛАНО] Подготовлены конфиги `slam_toolbox`, `AMCL`, `Nav2`, costmap и тестовая карта для интеграционных сценариев.
- [СДЕЛАНО] Реализованы интерфейсы `PatrolStatus`, `StartPatrol`, `StopPatrol`.
- [СДЕЛАНО] Реализован `mission_manager` с YAML-маршрутами, последовательной отправкой целей в `Nav2`, таймаутами, повторами и публикацией статуса миссии.
- [СДЕЛАНО] Добавлены unit-тесты для валидаторов, загрузчика маршрутов, интегратора одометрии и state machine.
- [СДЕЛАНО] Добавлены launch/integration тесты для `mock_bringup`, `slam`, `localization`, `navigation` и сценариев patrol через mock Nav2 action server.
- [СДЕЛАНО] Добавлены bash-скрипты сборки, запуска, тестирования, сохранения карты и проверки форматирования.
- [СДЕЛАНО] Подготовлены README, архитектурная документация, runbook, описание логирования, настройка Raspberry Pi и документ со структурой проекта.
- [СДЕЛАНО] Подробно прокомментированы ключевые C++ модули `patrolbot_base`, `patrolbot_mission_manager` и `patrolbot_utils`, включая runtime-логику, state machine, одометрию и загрузку маршрутов.
- [СДЕЛАНО] Подробно прокомментированы `launch`-профили bringup, `robot_state_publisher.launch.py`, описание робота в `URDF/Xacro` и ключевые system tests с mock Nav2 action server.
- [СДЕЛАНО] Подробно прокомментированы YAML-конфиги базы, mock LiDAR, `slam_toolbox`, `AMCL`, `Nav2`, рабочая миссия patrol и тестовые YAML-сценарии.
- [В РАБОТЕ] Поэтапное расширение подробных комментариев на оставшиеся второстепенные тесты, карты, вспомогательные файлы и дополнительную эксплуатационную документацию.
- [В РАБОТЕ] Адаптация `serial`-драйвера под реальный моторный контроллер.
- [НЕ СДЕЛАНО] Полевая валидация на реальном Raspberry Pi, LiDAR и моторном контроллере.
- [ПЛАНИРУЕТСЯ] Переход на полноценный аппаратный драйвер или `ros2_control` после стабилизации электроники.
- [ПЛАНИРУЕТСЯ] Добавление IMU и повторная оценка необходимости `robot_localization`.
- [ПЛАНИРУЕТСЯ] Расширение прикладной миссии сценариями расписаний, зон патруля и реакций на события.
